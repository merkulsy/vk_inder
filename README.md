# VKinder — бот для знакомств ВКонтакте

Бот ищет пользователей ВКонтакте по возрасту, полу и городу, показывает анкеты с тремя самыми популярными фотографиями и сохраняет избранное в PostgreSQL.


## Требования

- Python 3.9+
- PostgreSQL
- Токен группы ВКонтакте (для бота)
- Access token пользователя (для поиска людей и фото)


## Установка

1. Создайте базу данных PG:

```bash
createdb vk_inder_db
```

2. Установите зависимости:

```bash
pip install -r requirements.txt
```


## Настройка токенов

### Токен группы

См. инструкцию в файле [create_GROUP_TOKEN.md](create_GROUP_TOKEN.md)

### Access token пользователя

См. инструкцию в файле [create_ACCESS_TOKEN.md](create_ACCESS_TOKEN.md)


## Запуск

#### В PyCharm (рекомендуется)
Через зелёную кнопку **Run** (▶) для bot.py

#### Во вкладке **Terminal**
```bash
cd vk_inder
python bot.py
```

При запуске программа запросит:

- токен группы
- access token пользователя


## Остановка

#### PyCharm (рекомендуется)

Нажмите красную кнопку **Stop** (■) на панели Run.

#### Во вкладке **Terminal**

1. Кликните мышкой в терминал, чтобы он был активен.
2. Нажмите **Ctrl+C** один или два раза.
3. Если не помогло — кнопка Stop.


## Использование бота

1. Напишите сообщение в группу ВКонтакте.
2. Нажмите **Начать** — бот возьмёт из вашего профиля возраст, пол и город и начнёт поиск.
3. **Следующий** — показать следующую анкету.
4. **В избранное** — сохранить текущую анкету в БД.
5. **Избранные** — вывести список сохранённых людей.

В профиле VK должны быть указаны дата рождения и город.
